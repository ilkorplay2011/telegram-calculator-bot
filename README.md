# 🧮 Telegram Calculator Bot

Telegram-бот калькулятор с кнопочным и текстовым режимом ввода.

---

## ⚙️ Возможности

- 📱 Кнопочный калькулятор (как приложение)
- 💬 Текстовый ввод выражений
- ➕ Базовые операции: +, -, ×, ÷
- 🔢 Степени (^)
- 📊 Поддержка k / m / b:
  - k = 1000
  - m = 1_000_000
  - b = 1_000_000_000
- ⌫ удаление последнего символа
- 🧹 очистка выражения
- ➖/➕ смена знака
- 🧮 живой экран калькулятора

---

## 🚀 Установка
```bash
git clone https://github.com/your-repo/telegram-calculator-bot
cd telegram-calculator-bot

python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

pip install -r requirements.txt
```
## 🌟 Почему этот бот полезен

Этот проект создан как учебно-практический калькулятор-бот, который имитирует поведение реального приложения:

- работает как мини-калькулятор в Telegram
- поддерживает UI через кнопки
- умеет вычислять выражения как в обычном калькуляторе
- поддерживает сокращения (k, m, b)
- имеет гибридный режим (кнопки + текст)
